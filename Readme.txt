To run the web app 
1. start the frontend
	i)   go to  project-nexus-muge 
	ii)  then open cmd in that location 
	iii) run " npm install " command 
	iv)  run " npm run dev " command
2. start the backend
	i)   go to  server folder
	ii)  then open cmd in that location 
	iii) run " npm install " command 
	iv)  run " npm start " command
